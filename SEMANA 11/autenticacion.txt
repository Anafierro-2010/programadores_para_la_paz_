El login es el proceso mediante el cual un usuario se identifica para acceder a una aplicación. Esto permite verificar su identidad antes de utilizar determinadas funciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar cambios importantes. De esta manera se evita el acceso de personas no autorizadas. Nunca deben exponerse contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
